🚀 B.Sc Agri Rank 1 Dashboard

A personal study dashboard designed to help students prepare for the B.Sc Agriculture Entrance Examination (CCSHAU).

The project focuses on consistency, syllabus completion, performance tracking, and daily accountability.

🎯 Goal

Help a student systematically prepare for the B.Sc Agriculture Entrance Exam and maximize performance through:

Syllabus Tracking
Performance Analysis
Daily Goals
Study Streaks
Exam Countdown
Progress Monitoring
Motivation & Accountability
